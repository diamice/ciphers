## Шифр XOR

Шифр XOR — это простой алгоритм, который использует операцию XOR для шифрования и дешифрования данных. Для шифрования необходимо выбрать ключ, который будет применяться к каждому символу исходного текста.

**Файл:** `xor.py`

python xor.py
